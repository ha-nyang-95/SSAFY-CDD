# 🐆 B102_CHEETAH

## 📌 목적
이 저장소는 **팀원 간 카테고리별 자료조사를 체계적으로 공유**하기 위해 구성되었습니다.  
AI, Backend, Frontend, Embedded 영역별로 조사한 내용을 정리·관리하여 협업의 효율을 높이는 것이 목적입니다.

## 📁 폴더 구조

- **Embedded/** : 임베디드 시스템, 센서, 마이컴, 하드웨어 연동 관련 자료
- **WEB/** : 웹 사전조사 자료
  - **BE/** : Spring Boot 백엔드 프로토타입(`CrackTrack/`)
  - **FE/** : 프론트엔드 조사 자료
  - **webrtc/** : WebRTC 기반 드론 영상 스트리밍 PoC (`drone_py/`, `server_js/`, `index.html`)
  - **livecam/** : Node.js 기반 라이브캠 스트리밍 PoC

## 🗂️ 사용 방법

1. **자료 조사 후 카테고리에 맞는 폴더에 정리**해 주세요.
2. 파일명은 알아보기 쉽게 작성해 주세요. (예: `2025-07-11_AI_Model_Comparison.md`)
3. 조사한 내용은 Markdown 형식으로 작성해 주세요.
4. 새로운 카테고리가 필요하다면 팀원과 협의 후 추가해 주세요.

## 💬 팀 규칙

- 조사 내용의 출처를 반드시 남겨주세요.
- 정리 시 개인 의견과 조사 내용을 구분해 주세요.
- 매 주 주간 회의에서 각자 조사 내용을 간단히 공유합니다.
