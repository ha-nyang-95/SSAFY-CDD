### **프로젝트 기획 배경: 왜 CDD가 필요한가?**

기존의 구조물 안전 점검 방식은 여러 가지 문제점을 안고 있습니다.

* **고비용 및 비효율** : 사람이 직접 육안으로 검사하는 방식은 많은 시간, 비용, 그리고 인력이 투입됩니다.
* **안전사고의 위험** : 교량이나 댐처럼 위험한 환경에 작업자가 직접 노출되어야 하므로, 중대재해가 발생할 가능성이 항상 존재합니다.
* **점검의 한계** : 검사 주기가 길어 문제의 조기 발견이 어렵고, 점검 결과를 체계적으로 데이터화하여 관리하는 데에도 한계가 있습니다.

이러한 문제들을 해결하기 위해 저희는 AI와 드론 기술을 접목한 실시간 탐지, 정밀 분석, 그리고 3D 시각화 시스템을 고안하게 되었습니다.

### **저희의 솔루션: CDD (Crack Detection Drone) 소개**

CDD는 단순히 드론으로 촬영만 하는 것을 넘어, '현장을 함께 읽는 지능형 파트너'를 지향합니다. 기존 서비스와 비교했을 때 다음과 같은 차별점을 가집니다.

| 구분 | 기존 서비스 | **CDD (저희 솔루션)** |
| :--- | :--- | :--- |
| **탐지 방식** | 촬영 후, 별도의 후처리 분석 필요 | **실시간 탐지** 및 사용자의 즉각적인 판단 가능 |
| **검사 효율** | 사람이 직접 검사하여 시간과 비용 소모가 큼 | 실시간 탐색 및 데이터 자동 전송으로 **점검 속도 향상 및 비용 절감** |
| **데이터 활용** | 2D 사진, 목록 위주의 보고서 | **3D 모델 기반**의 직관적인 데이터 시각화 제공 |
| **위험 요소** | 높은 곳이나 좁은 공간 접근 시 사고 위험 존재 | 비접촉 비행 점검으로 **작업자 안전성 향상** |

### **CDD의 핵심 기능**

저희 CDD 시스템은 세 가지 핵심 기능을 통해 작동합니다.

1.  **실시간 균열 탐지**: 드론이 촬영하는 영상을 실시간으로 전송받아, 장치에 탑재된 AI(On-Device AI, YOLO)가 영상 속에서 균열을 즉시 탐지하고 사각형의 바운딩 박스로 표시해 줍니다.
2.  **선택형 정밀 검사**: 사용자는 실시간으로 탐지된 균열을 보고 정밀 검사가 필요하다고 판단할 수 있습니다. 이때 AI(Segmentation) 기술과 LiDAR 센서를 활용하여 균열의 깊이, 폭 등 상세 데이터를 정밀하게 측정합니다.
3.  **3D 시각화 리포트**: 드론이 탐사하며 수집한 영상들을 종합하여 구조물의 3D 모델을 생성합니다. 발견된 균열의 위치, 위험도, 정밀 측정 데이터를 3D 모델 위에 시각적으로 매핑하여 구조물의 전체적인 상태를 한눈에 파악할 수 있는 직관적인 리포트를 제공합니다.

### **시스템 구성 (Architecture)**

CDD 시스템은 다음과 같이 구성되어 있습니다.

* **드론** : 카메라와 LiDAR 센서를 장착하여 영상과 센서 데이터를 수집합니다.
* **온보드 시스템** : Jetson Orin Nano에서 1차적으로 AI 모델을 통해 실시간 균열 탐지를 수행합니다.
* **클라우드 서버 (AWS, Runpod)** : 전송된 영상과 이미지를 기반으로 3D 모델링 및 정밀 분석(Segmentation)을 수행하고, 데이터를 저장 및 관리합니다.
* **사용자 인터페이스** : 안전 관리자는 웹(shyo2.com)을 통해 드론을 조종하고, 탐지 결과를 실시간으로 확인하며, 3D 모델링된 최종 보고서를 확인할 수 있습니다.

### **기대 효과**

저희는 CDD 시스템 도입을 통해 다음과 같은 효과를 기대하고 있습니다.

* **비용 및 시간 절감** : 자동화된 탐사 및 분석을 통해 기존 방식 대비 90% 이상의 효율 증대를 목표로 합니다.
* **작업자 안전 확보** : 위험 지역을 비대면으로 점검하여 중대재해를 원천적으로 차단할 수 있습니다.
* **데이터 기반 유지보수** : 3D 데이터를 영구적으로 보존하고 변화 추이를 추적하여, 정량적이고 체계적인 보수 계획을 수립할 수 있습니다.
* **업무 효율성 증대** : 탐지된 균열 중 심각한 결함에만 집중하여 선택적으로 정밀 검사를 진행함으로써 전체 점검 프로세스를 최적화할 수 있습니다.
